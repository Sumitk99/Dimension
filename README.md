# Dimension App

A modern collaboration platform with chat, code, cloud, and deployment features. This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Features

- Modern UI with responsive design
- Chat interface for team collaboration
- Waitlist signup functionality
- Responsive navigation and footer

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory
- `Src/Components/` - React components
  - `Footer/` - Footer component
  - `Form/` - Chat interface component
  - `Hero/` - Hero section component
  - `Navbar/` - Navigation component
  - `ProductiveTeams/` - Features section component
  - `downhero/` - Secondary hero section component
